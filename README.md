# lines
Drawing fun lines with Logo and turtle graphics.

Run and tested on https://www.calormen.com/jslogo/ but, most probably, will work with other Logo interpreters.


## Examples

```logo
setsc [28 10 88]
setpencolor [19 60 49]
setpensize 3

spikey 100 10 2 8 3 5 10
```
